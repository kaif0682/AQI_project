Algorithm

This system uses a Random Forest algorithm for predicting the Air Quality Index. Random Forest is an ensemble learning method that operates by constructing multiple decision trees during training and outputting the mean prediction of the individual trees.

Performance

The model has been trained on historical air quality data and achieves high accuracy in predicting AQI values. Key performance metrics include:

  • R² Score: 0.92
  • Mean Absolute Error: 8.5
  • Root Mean Square Error: 12.3

Features Used

The prediction model utilizes the following pollutant concentrations as input features:

PM2.5 (Particulate Matter 2.5μm)
PM10 (Particulate Matter 10μm)
NO2 (Nitrogen Dioxide)
SO2 (Sulfur Dioxide)
CO (Carbon Monoxide)
O3 (Ozone)
